# Reference Board Builder_1

Первая версия заготовки Miro-приложения для сборки референс-доски.

Что уже есть:
- выбор локальной корневой папки через `webkitdirectory`
- разбор структуры `Категория / Подтип / Цвет / файлы`
- поддержка сценария без цветовых папок
- расчет раскладки фреймов по ширине и количеству рядов
- подготовка сортировки по цвету на основе анализа изображений
- интерфейс в стиле `image-align-tool`

Что еще не включено в этой версии:
- создание превью на доске
- создание финальных shape / frame / text / image в Miro

## Структура, которую ожидает приложение

```text
Референсы/
  Здания/
    Штукатурка/
      Белый/
        img-01.jpg
      Серый/
        img-02.jpg
    Кирпич/
      img-03.jpg
  Парки/
    Газоны/
      Зеленый/
        img-04.jpg
```

## Файлы
- `index.html` — entry page приложения
- `index.js` — открытие панели по клику на иконку
- `panel.html` — UI панели
- `app.js` — парсинг папок и расчет раскладки

## Как тестировать
1. Создай пустой статический проект.
2. Скопируй в него эти файлы.
3. Подними любую простую статику, например через VS Code Live Server.
4. Подключи URL `index.html` как entry point Miro app.
5. Открой панель, выбери папку и нажми:
   - `Проверить структуру`
   - `Проверить раскладку`
